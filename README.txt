This repository is a collection of test codes with Waveshare e-Paper ESP32 Driver Board.
The board is available from: http://www.waveshare.net/shop/e-Paper-ESP32-Driver-Board.htm
And here's the wiki page: http://www.waveshare.net/wiki/E-Paper_ESP32_Driver_Board

Tested with:
Waveshare 2.9 inch e-Paper display, switch to 'A', product link: http://www.waveshare.net/shop/2.9inch-e-Paper.htm
WF0213T50CZ1500E4, WUXI VISION PEAK TECHNOLOGY, 2.13 inch R/B/W e-Paper display, switch to 'B'
